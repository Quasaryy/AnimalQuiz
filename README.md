# Tech Stack
- **Language:** SWIFT 5
- **IDE:** Xcode 14
- **iOS:** 16
- **Archetecture:** MVC
- **Frameworks:**  UIKit, Foundation
- **Layout:** Interface Builder
- **Features:** 
  - DataModel
  - Progress Bar
  - Switchers
  - Slider
  - Segues
  - For all questions, only one ViewController was used without transitions
  - And many more
